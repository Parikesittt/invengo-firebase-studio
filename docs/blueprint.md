# **App Name**: InventarisKu

## Core Features:

- Product Entry: Add new products to the inventory, including details such as name, description, and initial stock.
- Stock Update: Update the stock levels when products are added (stock in) or sold (stock out).
- Inventory Listing: View the current inventory levels with a filter.
- Low Stock Alerts: Generate notifications when stock levels are low to prevent stockouts.
- Inventory Analysis: AI tool to predict low stock based on historic trends

## Style Guidelines:

- Primary color: Soft, muted green (#90EE90) to provide a sense of calm and organization.
- Background color: Light white (#F5F5F5), nearly desaturated green, to maintain a clean and spacious feel.
- Accent color: A slightly more saturated, brighter green (#7CFC00), still in the green family, for key interactive elements to draw attention.
- Body and headline font: 'PT Sans', a humanist sans-serif for clarity and readability.
- Simple, minimalist icons for navigation and actions.
- Clean, intuitive layout with a focus on usability. Consistent spacing and alignment to ensure ease of use.
- Subtle transitions for stock updates to provide a smooth user experience.
